# Planetary Glyph SVGs

These SVG glyphs are vendored from Wikimedia Commons fixed-width astronomical
and astrological symbols.

Core planetary symbols:
- Author: Denis Moskowitz
- License: Creative Commons Attribution-ShareAlike 4.0 International
- Source category: https://commons.wikimedia.org/wiki/Category:Fixed_width_astronomical_and_astrological_symbols

Node and Chiron symbols:
- Author: Kwamikagami
- License: Creative Commons Attribution-ShareAlike 4.0 International
- Source category: https://commons.wikimedia.org/wiki/Category:Fixed_width_astronomical_and_astrological_symbols

Black Moon Lilith:
- Author: unknown
- License: Public domain, simple geometry
- Source: https://commons.wikimedia.org/wiki/File:Astrological_symbol_for_Black_Moon.svg

All glyphs are rendered as local inline SVG so chart symbol sizing does not
depend on platform font fallback.
