Sky background asterism assets

`public/data/sky-asterisms.generated.json` and the WebP files under
`public/sky/illustrations/` are generated by:

```bash
node scripts/build-sky-asterisms.mjs
```

Sources:

- Asterism line data and constellation illustrations: Stellarium western sky
  culture. Text/data are CC BY-SA; illustrations are Free Art License.
- Star coordinates: compact J2000 Hipparcos conversion from
  `gmiller123456/hip2000`.

The app renders these assets with very low opacity as decorative context behind
the active chart surface.
