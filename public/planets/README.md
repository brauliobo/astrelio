Planet image assets

These raster assets are downsampled to a maximum 720px dimension for the sky
background renderer. Source imagery comes from NASA, NASA/JPL, NASA/SDO, and
NASA-derived public-domain Wikimedia Commons files.

- sun.jpg: NASA/SDO latest 193A image
- mercury.jpg: NASA Messenger/Mariner-derived Mercury image
- venus.jpg: NASA Venus image
- earth.jpg: NASA Apollo 17 Earth image
- moon.jpg: Moon image used as the sky Moon texture
- mars.jpg: NASA/ESA Mars image
- jupiter.jpg: NASA/JPL Jupiter image
- saturn.jpg: NASA/JPL/Cassini Saturn image
- uranus.jpg: NASA/JPL Uranus image
- neptune.jpg: NASA/Voyager Neptune image
- pluto.jpg: NASA/New Horizons Pluto image
